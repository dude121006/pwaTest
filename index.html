UPDATE: AltStore: This is a third-party solution that helps simplify the process of sideloading and refreshing apps. It still uses the same 7-day certificate system but adds automation to refresh your apps before they expire.
You'll need:
  • A computer (Mac or Windows) running the AltServer software
  • Your computer and iOS device on the same WiFi network regularly (for refreshing)
  • To trust the certificates on your device
For creating your C++ app specifically for personal use:
  • Qt is still a good option as it supports both Windows and iOS
  • You can use SQLite embedded in your app for SQL functionality without needing a separate server
    • For iOS deployment, you'll compile using Xcode with your Qt project The main limitation is still the 7-day certificate expiration if you're not paying for an Apple Developer account, but if you don't mind reconnecting your device weekly or using AltStore's refresh system, you can deploy your personal C++ apps to iOS without paying Apple's annual fee.
